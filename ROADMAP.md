Архитектурное обоснование — [docs/openclaw-integration.md](./docs/openclaw-integration.md).

- [x] Платформо-нейтральный конверт сообщения + `capabilities` — #8
- [x] Персона в конфиг (`persona/`): per-surface стиль, раскрытие ИИ — опция — #9
- [x] Identity-слой: персоны, политики, слияние только явное (API) — #10
      (кнопка подтверждения склейки в Telegram — этап 2)
- [x] Реестр инстансов + маршрутизация (`instances.json`) — #12
- [x] Бонус из этапа 5: Dockerfile + docker-compose, валидация env, тесты (29, node:test),
      эскалация не-текстовых сообщений
